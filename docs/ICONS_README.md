# Viber App Icons

## Required Icons

This directory should contain the following icon files for the PWA and SEO:

### Main Icons
- **icon-192.png** (192x192) - Standard app icon
- **icon-512.png** (512x512) - Large app icon
- **apple-icon.png** (180x180) - Apple touch icon
- **favicon.ico** - Browser favicon

### Open Graph & Social
- **og-image.png** (1200x630) - Open Graph preview image

## How to Generate

You can generate these icons from a single high-resolution source image (at least 1024x1024) using:

1. **Online Tools:**
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator

2. **Design Software:**
   - Figma, Photoshop, GIMP, Canva

## Design Guidelines

- **Background:** Solid color or subtle gradient (match theme purple #8b5cf6)
- **Icon:** Clear, recognizable symbol (music note, sound wave, or "V" logo)
- **Padding:** 10% safe area around edges for maskable icons
- **Style:** Modern, minimal, high contrast

## Color Palette

Use Viber's brand colors:
- Primary: #8b5cf6 (Purple)
- Secondary: #ec4899 (Pink)
- Accent: #3b82f6 (Blue)
- Background: Black or dark purple gradient
